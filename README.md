# Arduino Temperature Sensor Project with KY-013

This Arduino project measures temperature using the KY-013 temperature sensor

# Table of Contents

- [Materials](#materials)
- [Image](#image)

# Materials

- 2 Led (Blue and Red)
- 2 Resistors 220 ohm
- KY-013 sensor module
- Breadboard
- 1 lcd 16x2 with I2c module
- Arduino Uno R3
- Jumper Cables
